
### - 1️⃣ What is the difference between var, let, and const?

   // var is the old way of declaring variables. It is function-scoped and can be redeclared and updated, which can sometimes cause bugs.

   // let is block-scoped, meaning it only works inside the block where it is defined. It can be updated but not redeclared in the same scope.

   // const is also block-scoped, but its value cannot be changed after it is assigned.

 ###   - 2️⃣ What is the spread operator (...)?
    The spread operator (...) is used to take out or spread the values from an array or object.

    It makes it easy to copy or combine data without writing extra code.



###    - 3️⃣ What is the difference between map(), filter(), and forEach()?

    These are array methods, but they are used for different purposes.

    // map() creates a new array by applying a function to each element.

    // filter() creates a new array with only the elements that pass a condition.

    // forEach() runs a function on each element but does not return a new array.



###   - 4️⃣ What is an arrow function?

    An arrow function is a shorter and simpler way to write a function in JavaScript.

    It uses the => symbol, which makes the code easier to read and write compared to normal functions.


###    - 5️⃣ What are template literals?

    Template literals are a simple way to write strings in JavaScript using backticks (`).

They make it easy to add variables inside a string by using ${} without needing extra code.
